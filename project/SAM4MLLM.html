SAM4MLLM: Enhance Multi-Modal Large Language Model for Referring Expression Segmentation

Yi-Chia Chen, Wei-Hua Li, Cheng Sun, Yu-Chiang Frank Wang, Chu-Song Chen
European Conference on Computer Vision (ECCV), 2024

Paper Code Poster

Abstract

We introduce SAM4MLLM, an innovative approach that integrates the Segment Anything Model (SAM) with Multi-Modal Large Language Models (MLLMs) for pixel-aware tasks. Our method enables MLLMs to learn pixel-level location information without requiring extensive modifications to the model architecture or additional tokens.

We propose an inquiry-based prompting strategy to locate appropriate input points for SAM, guided by MLLM reasoning. This unifies detailed visual grounding with powerful language generation in a single language-driven framework, while remaining lightweight and architecture-agnostic. Extensive experiments on public benchmarks validate the effectiveness of SAM4MLLM.
